## Free Plan

#### Best for

Individuals, students, and small teams who want to explore ZeroThreat with ongoing, but limited access.

The Free Plan is a long-term free tier, not a time-limited trial. It allows basic scanning and platform exploration, but with limited report visibility and without premium automation or enterprise controls.

#### Included

* 5 scans on signup with limited report view
* 1 scan per month with limited report view
* Unlimited hosts
* 1 parallel scan per host
* Agentic AI Pentesting

#### Not Included

* Full report visibility
* Re-tests
* CI/CD integrations
* Issue tracking integrations
* Notification integrations
* Production-safe testing
* Complex UI flow automation
* Custom attack templates
* White-label reports

When a vulnerability is identified, the complete HTTP request and response details are shown only for the first occurrence. Additional instances of the same vulnerability are redacted. This same restriction extends to other security findings including outdated packages, SSL/certificate vulnerabilities, and cloud misconfigurations, which also appear partially hidden. In essence, this format proves the scanner can identify security issues in your application, but withholds comprehensive details across all findings which are reserved for paid plan users.</p>



## Professional Plans

Professional plans are built for security teams that need either predictable cloud-based scanning or flexible usage-based testing.

### Professional Cloud

**Best for**

Small to mid-sized security teams running recurring scans with predictable pricing.

**Highlights**

* 1 to 10 hosts
* 1 parallel scan per host
* Unlimited re-tests
* Notification integrations
* Agentic AI Pentesting
* Access to scan profile based advanced coverage options

**Not Included**

* CI/CD integrations
* Issue tracking integrations
* Custom attack templates using Nuclei and Burp
* Static IP
* Production-safe testing
* Complex UI flow automation
* White-label reports


### Professional Pay-Per-Scan

**Best for**

Teams that scan occasionally, run ad-hoc assessments, or prefer credit-based usage instead of a subscription.

**Highlights**

* Unlimited hosts
* Usage-based scan model
* Free 1-week re-test
* Agentic AI Pentesting
* Popular CVE visibility in scan profile selection

**Not Included**

* Notifications
* CI/CD integrations
* Issue tracking integrations
* Production-safe testing
* Complex UI flow automation
* Custom attack templates
* White-label reports


## Enterprise Plans

Enterprise plans are designed for organizations that need scale, governance, deployment flexibility, advanced integrations, branded reporting, and higher operational control.

### Enterprise Cloud

**Best for**

Organizations running security testing continuously across larger environments with automation and governance needs.

**Highlights**

* 10+ hosts
* 3 parallel scans per host
* Unlimited re-tests
* CI/CD integrations
* Issue tracking integrations
* Notification integrations
* Optional static IP
* Production-safe testing
* Agentic AI Pentesting
* Complex UI flow automation using Playwright
* Custom attack templates using Nuclei and Burp with faster updates
* White-label reports
* Full scan profile feature access (except Popular CVEs)


### Enterprise Pay-Per-Scan

**Best for**

Service providers, consulting teams, and enterprises that need enterprise-grade testing with a usage-based model.

**Highlights**

* Unlimited hosts
* Usage-based scan model
* Free 1-week re-test
* Optional static IP
* Agentic AI Pentesting
* Custom attack templates using Nuclei and Burp with faster updates
* White-label reports
* Scan profile access for tech stack based vulnerability and prebuilt attack templates

**Not Included**

* Notifications
* CI/CD integrations
* Issue tracking integrations
* Production-safe testing
* Complex UI flow automation
* Quick Test


### Enterprise On-Prem

**Best for**

Organizations that require private deployment and tighter infrastructure control.

**Highlights**

* Unlimited hosts
* 2 parallel scans per instance
* Unlimited re-tests
* CI/CD integrations
* Issue tracking integrations
* Notification integrations
* Production-safe testing
* Agentic AI Pentesting
* Complex UI flow automation using Playwright
* Custom attack templates using Nuclei and Burp with faster updates
* White-label reports
* Broad scan profile access


## Which Plan Should You Choose?

* Choose **Free Plan** for basic long-term access with limited report visibility.
* Choose **Professional Cloud** for recurring scans with predictable billing.
* Choose **Professional Pay-Per-Scan** for occasional usage with credits.
* Choose **Enterprise Cloud** for full cloud capability with scale and automation.
* Choose **Enterprise Pay-Per-Scan** for enterprise-grade testing in a credit-based model.
* Choose **Enterprise On-Prem** for private deployment and maximum infrastructure control.

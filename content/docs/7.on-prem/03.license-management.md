## License Management

This section explains how to manage your On-Prem license after installation, including deassociating a license and adding new targets or credits.



## When Deassociation Is Required

You must deassociate your license if:

* You plan to move the On-Prem installation to another machine
* You need to reinstall On-Prem from scratch
* The current server is being retired or replaced

Deassociation ensures that the license can be safely reused.



## Deassociate License from On-Prem

1. Log in to your **ZeroThreat Cloud account**
2. Navigate to your organization’s license (<img src="/Images/image%20(444).png" alt="" data-size="line">) section. This can be accessed via organization settings. 
3. Click on the **Deassociate Licenses** button.

::fiqure-img{source="/Images/image (445).png"}
::

4. Copy the generated De-Association token.

::fiqure-img{source="/Images/image (446).png"}
::

5. Return to the On-Prem CLI:

* Select **Deassociate License & System**

::fiqure-img{source="/Images/image (447).png"}
::

* Paste the token when prompted

::fiqure-img{source="/Images/image (448).png"}
::

This detaches the license from the current installation.

::hint{ style="background-color:#fff4e6; color:#f97316; border:1px solid #f6d2b0;" icon="triangle-exclamation" iconClass="text-[#f97316]"}

###### Failing to de-associate the license before reinstalling on another machine will result in a blocked state or error during the license activation process.
::



## Adding New Targets or Credits

To add targets or credits to your existing on-prem installation, you will need to purchase them from the Cloud version. Once done:

1. Log in to your **On-Prem ZeroThreat** URL
2. Navigate to your organization’s license (<img src="/Images/image%20(444).png" alt="" data-size="line">) section. This can be accessed via organization settings.

::fiqure-img{source="/Images/image (449).png"}
::

3. Click **Activate License** (<img src="/Images/image%20(466).png" alt="" data-size="line">)
4. Enter the license key for the top-up. You will need to paste the license key generated in your licenses section in your **ZeroThreat Cloud account** organization.
::fiqure-img{source="/Images/image (450).png"}
::

Once Activated you will see a successful activation screen.\
\
The newly purchased capacity will be added in your On-Prem local installation.

::fiqure-img{source="/Images/image (451).png"}
::
